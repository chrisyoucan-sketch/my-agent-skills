# Layout Rules

## Core Principle

Aim for balanced information density, not maximum content packing.
Each slide should look intentionally composed at a glance before the viewer reads any text.

## Density Bands

Classify slide fill into three bands using usable content area rather than the full page:

- `underfilled`: content uses less than about 35 percent of the usable area
- `balanced`: content uses about 35 to 72 percent of the usable area
- `overfilled`: content uses more than about 72 percent of the usable area

Use the density band only as a guide.
Also consider line length, paragraph count, and visual congestion.

## Balance Rules

- Keep the primary title region clearly separated from the body region.
- Preserve one dominant focal block per slide.
- Avoid stacking many unrelated small elements around the page.
- Keep similar objects visually consistent in size and alignment.
- Use asymmetry only when it is deliberate and still stable.

## Blank Space Rules

- Whitespace should frame the content, not strand it.
- Large empty regions are acceptable on `cover` and `section` slides.
- On regular content slides, excessive blank space usually means the structure is too weak or the content is too fragmented.

## Overcrowding Rules

- Do not compress margins to save a slide.
- Do not shrink all text uniformly when only one region is overloaded.
- Do not allow a slide to contain several long paragraphs plus charts plus notes.
- Split the slide when two or more major regions compete for attention.

## Preferred Corrections

When a slide is crowded, correct it in this order:

1. split into multiple slides
2. switch to a better slide type
3. reduce low-priority copy
4. restructure prose into bullets, labels, or cards
5. reduce body text size within the allowed range

When a slide is too empty, correct it in this order:

1. enlarge the focal region
2. tighten the composition around a clearer axis
3. strengthen title or section framing
4. add supporting context only if it improves comprehension
