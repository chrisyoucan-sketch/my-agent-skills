# Spacing System

## Safety Margin

For a 16:9 slide, keep all important content inside an outer safety margin of about `0.5 in` to `0.75 in`.
Use `0.6 in` as the default hard margin in automated checks.

## Gutter

Use consistent gutters between major columns or cards.
Recommended default gutter:

- `0.24 in` to `0.4 in` for dense business slides
- `0.4 in` to `0.6 in` for lighter editorial slides

## Vertical Rhythm

Keep one spacing scale across the slide instead of arbitrary gaps.
Recommended baseline scale:

- `8 pt`
- `12 pt`
- `16 pt`
- `24 pt`
- `32 pt`

Choose a base interval and reuse it.
Use larger spacing only at major region boundaries.

## Minimum Gaps

Preserve at least these clear gaps:

- title to body region: `0.20 in`
- text block to text block: `0.12 in`
- shape or image to text block: `0.16 in`
- card to card: `0.18 in`

If the slide type is intentionally airy, increase these distances rather than adding random blank space elsewhere.

## Alignment Rules

- Align repeated objects to shared left edges.
- Center only when the whole composition is center-driven.
- Match top edges for peer cards or columns.
- Match baselines for labels or short metrics when possible.
- Avoid tiny one-off offsets that make elements appear almost but not quite aligned.
