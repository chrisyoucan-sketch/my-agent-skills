# Deck Builder

Use this reference to convert raw business context into a consultant-style deck.

## Inputs

Expect some or all of:
- project notes
- meeting transcripts
- background documents
- problem statement
- deck type
- audience and decision needed

If deck type is missing, infer the closest fit from the context:
- management or steering deck: usually 8-12 slides
- sales or proposal deck: usually 5-8 slides
- discovery or diagnostic readout: usually 6-10 slides

## Process

### 1. Analyze context

Extract:
- situation: current state
- complication: tension, risk, or opportunity
- resolution: recommendation or next move
- evidence: facts, metrics, examples, quotes, cases

### 2. Determine structure

Create a slide sequence that follows SCR:
- opening: establish current state and stakes
- middle: surface the problem or opportunity with evidence
- closing: resolve the issue and make the decision path clear

Group related slides together.
Do not let one topic dominate without a reason.

### 3. Write the ghost deck

Write one action title per slide.
Each title must:
- be a complete sentence
- communicate the so-what
- use active voice
- stay under 15 words
- use specific numbers when supported by the source

### 4. Self-check before showing the deck

For each title, ask:
- Is this a sentence rather than a label?
- Would an executive care?
- Is the implication explicit?
- Is the wording concrete?
- Is the wording short enough for a slide title?

If any answer is no, rewrite the title before presenting it.

### 5. Present the ghost deck

Output a numbered table with:
- slide number
- action title
- confidence score from 0-100

Then add:
- a 3-4 sentence narrative that the titles tell in sequence
- a short note on any titles you changed during internal review

### 6. Write slide content

For each slide, produce:
- 3-5 bullets that prove the title
- one callout with the single most important insight
- 2-3 sentences of speaker notes
- a source tag for any factual claim

### 7. Final check

Before finalizing, verify:
- every bullet proves the title
- every number is traceable
- all slides contribute to the decision
- the deck closes with a clear recommendation or call to action
